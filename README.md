# Social Media Big Data Analyzer

A Streamlit-based Big Data application that analyzes trending topics from
Twitter and Reddit using TF-IDF and WordClouds.

## Features
- Twitter scraping using snscrape
- Reddit API integration
- TF-IDF keyword importance
- WordCloud visualization
- Streamlit UI with Tabs

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
